Clínica Atlantis

Esta aplicación fue creada como un proyecto de estudios del Instituto.

Es una aplicación web administrativa para una clinica, donde permite registrar una cita para el paciente, el mantenimiento de empleados, doctores, usuarios, etc.

Detalles:
- Se usó el lenguaje de programación Java y Spring Tool Suite.
- Se usó MySql como base de datos.
- La App web consumía una API REST.
- La API fue creada en Java, MySql, y se usó Postman para las consultas.
